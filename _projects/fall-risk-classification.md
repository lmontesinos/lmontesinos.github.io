This project investigates machine learning methods for classifying fall risk in older adults using quantitative assessment data.

The work aims to identify promising computational approaches that can support objective, scalable, and data-driven fall-risk assessment.

## Focus

- Feature extraction and signal analysis
- Supervised machine learning
- Balance and mobility assessment
- Translational potential for rehabilitation and preventive care
