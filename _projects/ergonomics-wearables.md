This project line uses wearable sensors, video-based ergonomic analysis, and functional performance measures to study work practices and their implications for safety, comfort, and productivity.

## Focus

- Ergonomic assessment
- EMG and IMU-based monitoring
- Functional task performance
- Evidence-based recommendations for workplace improvement
